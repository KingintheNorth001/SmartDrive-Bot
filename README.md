# SmartDrive-Bot
This project presents the design and implementation of a vision-based autonomous mobile robot capable of performing lane following and obstacle avoidance in a structured indoor environment. The system integrates computer vision, sensor-based perception, and closed loop control to achieve reliable navigation.


🔍 **What it does:**
• Detects road lanes using computer vision (OpenCV)
• Follows the lane using PID-based control
• Detects obstacles in real-time using ultrasonic sensors
• Makes intelligent decisions with a state machine
• Performs smooth lane switching and recovery

🧠 **Key Highlights:**
• Robust lane detection even with partial visibility
• Real-time obstacle avoidance
• Smooth and stable navigation
• Modular and scalable system design
